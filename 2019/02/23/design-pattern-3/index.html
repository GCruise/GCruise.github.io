单例模式的定义

单例模式保证整个系统中只有一个实例，并且自行实例化为整个系统提供这个实例。
单例模式需要满足以下三个条件：

  • 单例类只能有一个实例
  • 单例类必须自己创建自己唯一的实例
  • 单例类必须给其他所有对象提供这一实例
    在获取单例模式的时候，要保证不能产生多个实例对象。在使用单例模式时，也要考虑到实例对象可能会被多个线程同时使用，需要考虑线程安全问题。
    这里介绍几种实现单例模式的方法。
